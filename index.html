    Haikyuu Elastics Search

    This project is part of the Information Retrieval and Storage (ITCS414) course, which focuses on creating the Haikyuu Character Search System using Elasticsearch and the quicksort algorithm. The system is designed to provide a fast and user-friendly way to search for Haikyuu characters even when users forget the exact names.

    I was responsible for compiling character data into a JSON file and designing the front-end (index.html and search.html) for a smooth user experience. All team members were also required to understand the Python script and set up Elasticsearch and Kibana to run the project locally.
